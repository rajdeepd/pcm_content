# AP Practice Problems

### 1. Find 'd' if $t_n = 5n + 3$

The common difference ($d$) is the difference between any two consecutive terms.
* $t_1 = 5(1) + 3 = 8$
* $t_2 = 5(2) + 3 = 13$
* $d = t_2 - t_1 = 13 - 8 = 5$

**Answer:** $d = 5$

---

### 2. If $m \cdot t_m = n \cdot t_n$, find $t_{m+n}$

We are given $m[a + (m-1)d] = n[a + (n-1)d]$.
$am + m^2d - md = an + n^2d - nd$
$am - an + m^2d - n^2d - md + nd = 0$
$a(m - n) + d(m^2 - n^2) - d(m - n) = 0$
$a(m - n) + d(m - n)(m + n) - d(m - n) = 0$

Assuming $m \neq n$, we can divide by $(m - n)$:
$a + d(m + n) - d = 0$
$a + (m + n - 1)d = 0$

By definition, $t_{m+n} = a + (m + n - 1)d$.
**Answer:** $t_{m+n} = 0$

---

### 3. If $t_m = \frac{1}{n}$ and $t_n = \frac{1}{m}$, find $t_{mn}$

We have a system of equations:
1.  $a + (m - 1)d = \frac{1}{n}$
2.  $a + (n - 1)d = \frac{1}{m}$

Subtract (2) from (1):
$(m - 1 - n + 1)d = \frac{1}{n} - \frac{1}{m}$
$(m - n)d = \frac{m - n}{mn}$
$d = \frac{1}{mn}$

Substitute $d$ back into (1):
$a + (m - 1)\frac{1}{mn} = \frac{1}{n}$
$a + \frac{m}{mn} - \frac{1}{mn} = \frac{1}{n}$
$a + \frac{1}{n} - \frac{1}{mn} = \frac{1}{n}$
$a = \frac{1}{mn}$

Now, find $t_{mn}$:
$t_{mn} = a + (mn - 1)d = \frac{1}{mn} + (mn - 1)\frac{1}{mn}$
$t_{mn} = \frac{1}{mn} + \frac{mn}{mn} - \frac{1}{mn} = \frac{1}{mn} + 1 - \frac{1}{mn}$

**Answer:** $t_{mn} = 1$

### 4. If $\frac{t_4}{t_7} = \frac{2}{3}$, find $\frac{t_6}{t_8}$

Given $\frac{a + 3d}{a + 6d} = \frac{2}{3}$
$3(a + 3d) = 2(a + 6d)$
$3a + 9d = 2a + 12d$
$a = 3d$

Now, we need to find $\frac{t_6}{t_8} = \frac{a + 5d}{a + 7d}$.
Substitute $a = 3d$:
$\frac{3d + 5d}{3d + 7d} = \frac{8d}{10d} = \frac{8}{10}$

**Answer:** $\frac{t_6}{t_8} = \frac{4}{5}$

### 5. Find the number of identical terms

**AP 1:** $2, 5, 8, \ldots$ (60 terms)
* $a_1 = 2$, $d_1 = 3$
* Last term $t_{60} = 2 + (60 - 1)3 = 2 + 177 = 179$

**AP 2:** $3, 5, 7, \ldots$ (50 terms)
* $a_2 = 3$, $d_2 = 2$
* Last term $t_{50} = 3 + (50 - 1)2 = 3 + 98 = 101$

The identical terms form a new A.P.
* **First term (by inspection):** AP 1: (2, 5, 8, 11...). AP 2: (3, 5, 7, 9, 11...). The first identical term is 5. So, $a_{new} = 5$.
* **New common difference:** $d_{new} = \text{LCM}(d_1, d_2) = \text{LCM}(3, 2) = 6$.
* The identical terms are $5, 11, 17, \ldots$

The terms in this new A.P. must be less than or equal to the *minimum* of the two last terms.
$\text{Max value} = \min(179, 101) = 101$.

Let $n$ be the number of identical terms. We find $n$ such that $t_n \le 101$:
$a_{new} + (n - 1)d_{new} \le 101$
$5 + (n - 1)6 \le 101$
$(n - 1)6 \le 96$
$n - 1 \le 16$
$n \le 17$

**Answer:** 17

### 6. Polygon interior angles

The sum of the interior angles of an $n$-sided polygon is $S_n = (n - 2) \times 180$.
The sum of the $n$ angles in A.P. is $S_n = \frac{n}{2}[2a + (n - 1)d]$.
Given $a = 120^{\circ}$ and $d = 5^{\circ}$.

Equating the two formulas:
$\frac{n}{2}[2(120) + (n - 1)5] = (n - 2)180$
$\frac{n}{2}[240 + 5n - 5] = 180n - 360$
$\frac{n}{2}[235 + 5n] = 180n - 360$
$235n + 5n^2 = 360n - 720$
$5n^2 - 125n + 720 = 0$
$n^2 - 25n + 144 = 0$
$(n - 9)(n - 16) = 0$
So, $n = 9$ or $n = 16$.

We must check both solutions. An interior angle of a convex polygon must be $< 180^{\circ}$.
* **Case 1: $n = 9$**
    Largest angle $t_9 = a + 8d = 120 + 8(5) = 120 + 40 = 160^{\circ}$. This is valid.
* **Case 2: $n = 16$**
    Largest angle $t_{16} = a + 15d = 120 + 15(5) = 120 + 75 = 195^{\circ}$. This is invalid.

**Answer:** 9 sides

---

### 7. Proof

Let $d$ be the common difference. $d = a_2 - a_1 = a_3 - a_2 = \ldots = a_k - a_{k-1}$.
Consider the general term $\frac{1}{a_{k-1} a_k}$.
We can rewrite this using partial fractions:
$\frac{1}{a_{k-1} a_k} = \frac{1}{d} \left( \frac{d}{a_{k-1} a_k} \right) = \frac{1}{d} \left( \frac{a_k - a_{k-1}}{a_{k-1} a_k} \right) = \frac{1}{d} \left( \frac{1}{a_{k-1}} - \frac{1}{a_k} \right)$

Let $S = \frac{1}{a_1 a_2}+\frac{1}{a_2 a_3}+\ldots+\frac{1}{a_{n-1} a_n}$.
$S = \sum_{k=2}^{n} \frac{1}{a_{k-1} a_k} = \sum_{k=2}^{n} \frac{1}{d} \left( \frac{1}{a_{k-1}} - \frac{1}{a_k} \right)$
$S = \frac{1}{d} \left[ \left(\frac{1}{a_1} - \frac{1}{a_2}\right) + \left(\frac{1}{a_2} - \frac{1}{a_3}\right) + \ldots + \left(\frac{1}{a_{n-1}} - \frac{1}{a_n}\right) \right]$

This is a telescoping series. All intermediate terms cancel out.
$S = \frac{1}{d} \left[ \frac{1}{a_1} - \frac{1}{a_n} \right]$
$S = \frac{1}{d} \left[ \frac{a_n - a_1}{a_1 a_n} \right]$

We know $a_n = a_1 + (n - 1)d$, which means $a_n - a_1 = (n - 1)d$.
Substitute this into the numerator:
$S = \frac{1}{d} \left[ \frac{(n - 1)d}{a_1 a_n} \right]$
$S = \frac{n-1}{a_1 a_n}$ (Q.E.D.)

### 8. Find $t_{22}$ if $t_5 = 17, t_7 = 15$

1.  $a + (5 - 1)d = 17 \Rightarrow a + 4d = 17$
2.  $a + (7 - 1)d = 15 \Rightarrow a + 6d = 15$

Subtract (1) from (2):
$(a + 6d) - (a + 4d) = 15 - 17$
$2d = -2 \Rightarrow d = -1$

Substitute $d = -1$ into (1):
$a + 4(-1) = 17$
$a - 4 = 17 \Rightarrow a = 21$

Now find $t_{22}$:
$t_{22} = a + (22 - 1)d = 21 + 21(-1) = 21 - 21$

**Answer:** $t_{22} = 0$

## 6\. Polygon Interior Angles

### ðŸ“ Problem

The interior angles of a polygon are in an arithmetic progression. The smallest angle is **$a = 120^{\circ}$** and the common difference is **$d = 5^{\circ}$**. Find the number of sides of the polygon.

-----

### Solution

1.  **Formulas for the Sum of Angles:**

      * The sum of interior angles for any $n$-sided polygon is given by:
        $$S_n = (n - 2) \times 180$$
      * The sum of $n$ terms in an arithmetic progression (A.P.) is:
        $$S_n = \frac{n}{2}[2a + (n - 1)d]$$

2.  **Equating the Formulas:**
    We can set the two formulas equal to each other, substituting the given values $a = 120$ and $d = 5$:
    $$(n - 2) \times 180 = \frac{n}{2}[2(120) + (n - 1)5]$$

3.  **Solving for $n$:**

      * Simplify the equation:
        $$180(n - 2) = \frac{n}{2}[240 + 5n - 5]$$
        $$180n - 360 = \frac{n}{2}[235 + 5n]$$
      * Multiply by 2 to clear the fraction:
        $$360n - 720 = n(235 + 5n)$$
        $$360n - 720 = 235n + 5n^2$$
      * Arrange into a quadratic equation ($ax^2 + bx + c = 0$):
        $$5n^2 + 235n - 360n + 720 = 0$$
        $$5n^2 - 125n + 720 = 0$$
      * Divide the entire equation by 5:
        $$n^2 - 25n + 144 = 0$$
      * Factor the quadratic equation:
        $$(n - 9)(n - 16) = 0$$
      * This gives two possible solutions for $n$:
        **$n = 9$** or **$n = 16$**

4.  **Checking the Solutions:**
    We must check if both solutions are valid. An interior angle of a convex polygon cannot be $180^{\circ}$ or greater. We check the largest angle for each case.

      * **Case 1: $n = 9$ (A 9-sided polygon)**
        The largest angle is the 9th term ($t_9$):
        $t_9 = a + (n - 1)d = 120 + (9 - 1)5$
        $t_9 = 120 + 8(5) = 120 + 40 = \mathbf{160^{\circ}}$
        This is a valid angle (less than $180^{\circ}$).

      * **Case 2: $n = 16$ (A 16-sided polygon)**
        The largest angle is the 16th term ($t_{16}$):
        $t_{16} = a + (n - 1)d = 120 + (16 - 1)5$
        $t_{16} = 120 + 15(5) = 120 + 75 = \mathbf{195^{\circ}}$
        This is **invalid**, as an interior angle of $195^{\circ}$ would make the polygon non-convex (it would "fold" in on itself).

### Final Answer

The only valid solution is **$n = 9$**. The polygon has **9 sides**.
