# Mind Map

A visual summary of the chapter's concepts.

## Key Concepts

### Definition
- Arithmetic Progression (AP)
- Common Difference ($d$)
- General Form: $a, a+d, a+2d, \ldots$

### Formulas

| Concept | Formula |
|---------|---------|
| $n$th Term | $a_n = a + (n-1)d$ |
| $n$th Term from End | $a_m = l - (m-1)d$ |
| Sum of $n$ Terms | $S_n = \frac{n}{2}[2a + (n-1)d]$ |
| Sum (with last term) | $S_n = \frac{n}{2}(a + l)$ |
| Sum of first $n$ integers | $\frac{n(n+1)}{2}$ |

### Key Properties
- If $a$, $b$, $c$ are in AP, then $b - a = c - b$ or $2b = a + c$
- $n$th term: $a_n = S_n - S_{n-1}$ (for $n > 1$)
- Common difference: $d = a_{n} - a_{n-1}$
