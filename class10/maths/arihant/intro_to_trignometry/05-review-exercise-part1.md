- **(b)** Both Assertion and Reason are correct but Reason is not the correct explanation of Assertion.

<a id="q25"></a>**25.** *[CBSE 2023 Standard]* [→ Answer](#a25)

**Assertion:** In right angled triangles $\triangle ABC$ and $\triangle DEF$ ($\angle C = \angle F = 90°$), $\angle B$ and $\angle E$ are acute angles such that $\sin B = \sin E$, then $\angle B = \angle E$.

<a id="q26"></a>**26.** [→ Answer](#a26)

<a id="q27"></a>**27.** *[CBSE 2023 Basic]* [→ Answer](#a27)

<a id="q28"></a>**28.** [→ Answer](#a28)

**Assertion:** $\cos^2 A - \sin^2 A = 1$ is a trigonometric identity.

**Reason:** An equation involving trigonometric ratios of an angle is called trigonometric identity, if it is true for all values of the angles involved.

## Answers

<a id="a1"></a>**1. (a)** Given, $\tan \theta = \frac{5}{12}$ [← Question](#q1)

By Pythagoras theorem: $AC^2 = AB^2 + BC^2 = 5^2 + 12^2 = 169$ → $AC = 13$

So $\sin \theta = \frac{5}{13}$ and $\cos \theta = \frac{12}{13}$

$\frac{\sin \theta + \cos \theta}{\sin \theta - \cos \theta} = \frac{\frac{5}{13} + \frac{12}{13}}{\frac{5}{13} - \frac{12}{13}} = \frac{17}{-7} = \frac{-17}{7}$

<a id="a2"></a>**2. (b)** In $\triangle ACB$, $\tan A = \frac{BC}{AC} = \frac{8}{7}$ [← Question](#q2)

Since $\angle C = 90°$, $\cot B = \frac{BC}{AC} = \frac{8}{7}$

<a id="a3"></a>**3. (c)** Given, $\cos \theta = \frac{1}{\sqrt{2}} = \cos 45°$ → $\theta = 45°$ [← Question](#q3)

$\tan \theta = \tan 45° = 1$

<a id="a4"></a>**4. (c)** In isosceles $\triangle ABC$, let $BP = h$ (perpendicular to AC) [← Question](#q4)

Area of semi-circle = Area of triangle: $\frac{1}{2}\pi r^2 = \frac{1}{2} \cdot 2r \cdot h$ → $h = \frac{\pi r}{2}$

$\tan x = \frac{h}{r} = \frac{\pi r/2}{r} = \frac{\pi}{2}$

<a id="a5"></a>**5. (a)** $\tan 30° = \frac{1}{\sqrt{3}}$ [← Question](#q5)

$\frac{2\tan 30°}{1 + \tan^2 30°} = \frac{2 \cdot \frac{1}{\sqrt{3}}}{1 + \frac{1}{3}} = \frac{\frac{2}{\sqrt{3}}}{\frac{4}{3}} = \frac{2}{\sqrt{3}} \times \frac{3}{4} = \frac{\sqrt{3}}{2} = \sin 60°$

<a id="a6"></a>**6. (b)** $\sin \theta - \cos \theta = 0$ → $\sin \theta = \cos \theta$ → $\tan \theta = 1$ → $\theta = 45°$ [← Question](#q6)

<a id="a7"></a>**7. (c)** If $\tan \theta + \cot \theta = 2$, then $\theta = 45°$ [← Question](#q7)

$\sin^3 45° + \cos^3 45° = \left(\frac{1}{\sqrt{2}}\right)^3 + \left(\frac{1}{\sqrt{2}}\right)^3 = \frac{1}{2\sqrt{2}} + \frac{1}{2\sqrt{2}} = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2}$

<a id="a8"></a>**8. (a)** $\cos \theta = \frac{\sqrt{3}}{2} = \cos 30°$ → $\theta = 30°$ [← Question](#q8)

$\sin \phi = \frac{1}{2} = \sin 30°$ → $\phi = 30°$

$\tan(\theta + \phi) = \tan 60° = \sqrt{3}$

<a id="a9"></a>**9. (d)** $\tan^2 45° - \cos^2 60° = 1 - \left(\frac{1}{2}\right)^2 = 1 - \frac{1}{4} = \frac{3}{4}$ [← Question](#q9)

<a id="a10"></a>**10. (d)** $\cot \theta = \frac{1}{\sqrt{3}} = \cot 60°$ → $\theta = 60°$ [← Question](#q10)

$\sec^2 60° + \csc^2 60° = 4 + \frac{4}{3} = \frac{16}{3} = 5\frac{1}{3}$

<a id="a11"></a>**11. (c)** In $\triangle PQR$, $\angle Q = 90°$ and $\angle P = 45°$, so $\angle R = 45°$ [← Question](#q11)

$\tan P - \cos^2 R = \tan 45° - \cos^2 45° = 1 - \left(\frac{1}{\sqrt{2}}\right)^2 = 1 - \frac{1}{2} = \frac{1}{2}$

<a id="a12"></a>**12. (d)** From the figure, in $\triangle PQR$, $\tan Q = \frac{PR}{QR} = \frac{4}{3}$ [← Question](#q12)

<a id="a13"></a>**13. (d)** Using Pythagoras in $\triangle PRS$, $RS = \frac{16}{3}\sqrt{3}$ units [← Question](#q13)

<a id="a14"></a>**14. (d)** $\cos^4 A - \sin^4 A = (\cos^2 A + \sin^2 A)(\cos^2 A - \sin^2 A) = 1 \cdot (\cos^2 A - \sin^2 A)$ [← Question](#q14)

$= \cos^2 A - (1 - \cos^2 A) = 2\cos^2 A - 1$

<a id="a15"></a>**15. (d)** Option (d) $\tan^2 \theta - \sec^2 \theta = 1$ is NOT true. [← Question](#q15)

The correct identity is $\sec^2 \theta - \tan^2 \theta = 1$

<a id="a16"></a>**16. (c)** From $x\sin \theta = y\cos \theta$, we get $x = y\cot \theta$ [← Question](#q16)

Substituting in the first equation and simplifying: $x^2 + y^2 = 1$

<a id="a17"></a>**17. (a)** From the identity $\sin^2 A + \cos^2 A = 1$ [← Question](#q17)

$1 - \cos^2 A = \sin^2 A$

<a id="a18"></a>**18. (d)** $\frac{\cos^2 \theta}{\sin^2 \theta} - \frac{1}{\sin^2 \theta} = \frac{\cos^2 \theta - 1}{\sin^2 \theta} = \frac{-\sin^2 \theta}{\sin^2 \theta} = -1$ [← Question](#q18)

<a id="a19"></a>**19. (d)** Using identity $\sec^2 \theta - \tan^2 \theta = 1$ [← Question](#q19)

$(\sec \theta - \tan \theta)(\sec \theta + \tan \theta) = 1$

$\frac{1}{3} \times (\sec \theta + \tan \theta) = 1$ → $\sec \theta + \tan \theta = 3$

<a id="a20"></a>**20. (b)** $(1 + \tan^2 A)(1 + \sin A)(1 - \sin A) = \sec^2 A \cdot (1 - \sin^2 A) = \sec^2 A \cdot \cos^2 A = 1$ [← Question](#q20)

<a id="a21"></a>**21. (b)** $\frac{(\sec^2 \theta - 1) \cos^2 \theta}{\sin \theta} = \frac{\tan^2 \theta \cdot \cos^2 \theta}{\sin \theta} = \frac{\sin^2 \theta}{\sin \theta} = \sin \theta = \frac{3}{4}$ [← Question](#q21)

<a id="a22"></a>**22. (b)** Given $\sec \theta + \tan \theta = p$ and using $\sec^2 \theta - \tan^2 \theta = 1$ [← Question](#q22)

$\sec \theta - \tan \theta = \frac{1}{p}$

Adding: $2\sec \theta = p + \frac{1}{p}$. Subtracting: $2\tan \theta = p - \frac{1}{p} = \frac{p^2 - 1}{p}$

$\tan \theta = \frac{p^2 - 1}{2p}$

<a id="a23"></a>**23. (c)** Given $\sin \theta = \frac{a}{b}$, using $\sin^2 \theta + \cos^2 \theta = 1$ [← Question](#q23)

$\cos^2 \theta = 1 - \frac{a^2}{b^2} = \frac{b^2 - a^2}{b^2}$ → $\cos \theta = \frac{\sqrt{b^2 - a^2}}{b}$

<a id="a24"></a>**24. (b)** $p^2 - q^2 = (a \cot \theta + b \csc \theta)^2 - (b \cot \theta + a \csc \theta)^2$ [← Question](#q24)

$= (a^2 - b^2)\cot^2 \theta + (b^2 - a^2)\csc^2 \theta = (b^2 - a^2)(\csc^2 \theta - \cot^2 \theta) = b^2 - a^2$

<a id="a25"></a>**25. (a)** Both Assertion and Reason are correct and Reason is the correct explanation. [← Question](#q25)

If $\sin B = \sin E$ in right triangles, the triangles are similar (AAA), hence $\angle B = \angle E$.

<a id="a26"></a>**26. (a)** Both Assertion and Reason are correct and Reason is the correct explanation. [← Question](#q26)

Since $\sec^2 \theta \geq 1$, we need $\frac{4xy}{(x+y)^2} \geq 1$. This implies $(x-y)^2 \leq 0$, which is only true when $x = y$.

<a id="a27"></a>**27. (a)** Both Assertion and Reason are correct and Reason is the correct explanation. [← Question](#q27)

$(\csc \theta - \cot \theta)(\csc \theta + \cot \theta) = \csc^2 \theta - \cot^2 \theta = 1$

Hence they are reciprocals of each other.

<a id="a28"></a>**28. (d)** Assertion is incorrect but Reason is correct. [← Question](#q28)

The correct identity is $\sin^2 A + \cos^2 A = 1$, not $\cos^2 A - \sin^2 A = 1$.

<a id="a29"></a>**29.** [← Question](#q29)

**(i)** Yes. $\sin A = \frac{\text{Perpendicular}}{\text{Hypotenuse}}$. In $\triangle PQA$, the perpendicular $PQ$ is greatest relative to its hypotenuse compared to the other triangles, so $\sin A$ is greatest.
